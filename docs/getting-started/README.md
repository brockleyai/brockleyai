# Getting Started

New to Brockley? Start here. This section walks you from zero to a running graph in minutes, then explains the architecture underneath.

## Reading order

1. **[What is Brockley?](introduction.md)** -- What Brockley does, how it differs from other agent tooling, and the key concepts you'll encounter throughout the docs.

2. **[Quickstart](quickstart.md)** -- Clone the repo, start the stack with `make dev`, deploy your first graph, and invoke it. Takes about five minutes.

3. **[Build Your First Graph](first-graph.md)** -- A hands-on walkthrough of creating a graph from scratch -- defining nodes, connecting edges, and running it.

4. **[Architecture Overview](architecture-overview.md)** -- How the pieces fit together: API server, workers, task queue, database, and the six peer interfaces.
